Projekt in Unity öffnen:
Unity Hub öffnen
Add / Open → Projektordner auswählen (der Ordner, in dem Assets/ liegt)
Projekt mit der passenden Unity-Version starten
Beim ersten Öffnen: Unity importiert automatisch Packages/Assets

Build erstellen
In Unity: File → Build Settings
Zielplattform auswählen --> Vision OS
Falls nötig: Switch Platform

Szene hinzufügen:
Add Open Scenes --> SampleScene, Room_Nature

Build klicken → Zielordner wählen
Ergebnis liegt im gewählten Build-Ordner
